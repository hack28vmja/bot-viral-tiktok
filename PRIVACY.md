# Política de Privacidad - Bot Viral TikTok

**Última actualización:** 2024

Esta aplicación ("el Bot") es una herramienta de automatización personal.

1. **Recopilación de datos:** El Bot no recopila, almacena ni comparte datos personales de terceros.
2. **Almacenamiento local:** Todos los tokens de acceso y credenciales se almacenan exclusivamente de forma local en la máquina del usuario mediante un archivo `.env`.
3. **Uso de Datos de TikTok:** El acceso a la cuenta de TikTok se utiliza únicamente para la subida automatizada de videos generados por el usuario.
4. **Terceros:** No se comparten datos con ninguna entidad externa, a excepción de las APIs oficiales utilizadas (Google Gemini, ElevenLabs, TikTok).
